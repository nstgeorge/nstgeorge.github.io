Instant Coffee  Created by Nate St. George

                        

                          Instant Coffee provides an easy way for programmers to create small tools for school, work, or other
                          environments where quick, advanced calculations may be useful. IC also provides helpful features to
                          streamline the process of creating a tool. 

                          Show/Out/Clear:  

                          The commands show, out, and clear provide the basis of console interaction in IC. Show will run JSON.stringify()
                          on the argument before printing in the console. This way, you can see the contents of objects instead of
                          [Object object], like what out returns. Out will simply print out the argument like console.log().
                          Clear clears the console. 

                          Units:  

                          IC provides a library of precalculated units and conversions. It currently only supports units of length.
                          However, these units are in the BigDecimal format, which allows for more accuracy in conversions like lightyears
                          to planck lengths. To see the list of all available units, run show units or show u. While the actual numeric
                          values are inaccessible in this way, you can still see the names of each unit. 

                          CDN Support:  

                          IC supports any code hosted on a CDN, including code you might write. This allows you to work as comfortably
                          as you can in the environment. To add a CDN, simply click the Frameworks and Plugins button and paste the
                          URL to the CDN in the box. To remove a CDN, click the X next to it. This will reload the page, but don't worry -
                          code written in IC is persistent and automatically saved on every keyboard press. Speaking of which... 

                          Saving and Loading:  

                          You can save and load your code easily in IC. Just click Save/Load Program and name it. JS files will have a
                          double slash before the name, while CoffeeScript will have a number sign/pound/hashtag/octothorpe. (Personally
                          I think octothorpe is the best name for the # symbol.)  IC will not automatically switch evaluation modes.
                          You must do this yourself.  

                          The UI Panel:  

                          CustomPanel allows you to easily create a UI for your program. Run 'out CustomPanel.help' for help with this
                          functionality.
